# 通知

通知类型有五种
1. 前置通知
2. 后置通知
3. 最终通知
4. 环绕通知
5. 异常抛出通知

前置通知：
1. 执行在目标对象方法之前
2. 使用< aop:before >标签
3. 应用于方法开始时校验

后置通知：
1. 执行在目标方法之后，有异常不执行
2. 使用< aop:after-returning >
3. 应用于修改方法的返回值

最终通知：
1. 执行在目标方法完成之后，在后置通知之后，有无异常都会执行
2. 使用< aop:after >标签
3. 应用于如释放资源

异常抛出通知：
1. 执行于异常抛出后
2. 使用< aop:after-throwing >标签
3. 应用于包装异常

环绕通知:
1. 相当于前四个通知的合体
2. 使用< aop:around >标签
3. 应用于事务、统计代码执行时机等
